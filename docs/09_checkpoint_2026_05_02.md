## Bloqueio Atual

O Google Chat mostra:

```text
Finance AI POC nao esta respondendo
```

Motivo mais provavel: o Cloud Run ainda esta privado. O Chat nao envia o mesmo token usado nos testes manuais.

Comando tentado:

```bash
gcloud run services add-iam-policy-binding finance-ai-chat-adapter \
  --project hml-data-clean \
  --region us-central1 \
  --member="allUsers" \
  --role="roles/run.invoker"
```

Erro:

```text
Permission 'run.services.setIamPolicy' denied
```

## Acao Necessaria

Pedir para um administrador do projeto `hml-data-clean` conceder `roles/run.invoker` para `allUsers` no servico `finance-ai-chat-adapter`, ou definir uma alternativa permitida pela politica corporativa.

Comando para o administrador:

```bash
gcloud run services add-iam-policy-binding finance-ai-chat-adapter \
  --project hml-data-clean \
  --region us-central1 \
  --member="allUsers" \
  --role="roles/run.invoker"
```
